# ✅ Django Todo Demo App

A simple, functional **Todo** application built with Django for learning and demonstration purposes.  
This project is ideal for beginners to understand Django’s MTV pattern, class‑based views, forms, and basic CRUD operations.

## 🚀 Features

- Create, Read, Update, Delete (CRUD) todo items
- Mark tasks as completed / pending
- Clean, responsive UI (Bootstrap optional)
- SQLite database (easily switchable to PostgreSQL/MySQL)
- Admin panel for data management

## 🛠️ Tech Stack

- **Backend:** Django 4.x / 5.x
- **Database:** SQLite (development)
- **Frontend:** HTML, CSS, Bootstrap 5 (or your own styling)
- **Language:** Python 3.10+

## 📋 Prerequisites

- Python 3.10 or higher installed
- Git installed
- (Optional) virtual environment tool like `venv`
